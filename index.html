Welcome to Humboldt Wireless Lab.

The Humboldt Wireless Lab is a Wireless Testbed at the campus of the Humboldt-University Berlin.

Getting started

Download the brn-tools repository and run the build script using "/bin/bash ./brn-tools.sh".
